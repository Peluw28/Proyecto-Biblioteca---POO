# Proyecto-Biblioteca---POO
Proyecto de biblioteca funcional en lenguaje de código C++. De Darío Bosiak y Juan Luis Bordón.
Al ejecutar el programa, aparece el menú principal en el cual se puede elegir una de estas opciones :

# --- I. Gestionar Items ---
- add-item -> Agregar un nuevo item al inventario  
- remove-item -> Eliminar un item existente  
- edit-item -> Modificar los datos de un item

# --- II. Gestionar Usuarios ---
- add-user -> Agregar un nuevo usuario  
- block-user -> Bloquear un usuario  
- unblock-user -> Desbloquear un usuario

# --- III. Préstamos ---
- loan -> Realizar el préstamo de un item a un usuario  
- return -> Realizar la devolución de un item

# --- IV. Búsqueda y Listado ---
- search -> Buscar items según el autor
- list -> Listar todos los items

# --- V. Reportes ---
- report -> Generar reportes de inventario, préstamos o usuarios

# --- VI. Exportación / Importación --- 
- export → Guardar los datos actuales en un archivo
- import → Cargar datos desde un archivo previamente guardado
! - Es necesario copiar el destino del archivo para importar / exportar - !
